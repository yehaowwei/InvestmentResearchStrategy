# BI 图表平台前端开发方案

## 1. 项目范围

本项目从零建设一套 BI 图表平台前端，范围限定为图表配置与展示，不包含后端查询引擎、权限系统、审批流程、任务调度等后端平台能力。

本期前端建设范围包括：

- 图表模板体系
- 设计态页面
- 运行态页面
- 折线图、柱状图、散点图、表格、混合图
- 图层配置与切换
- 统计指标配置与展示

## 2. 方案主线

本项目从配置视角组织前端能力。前端的核心不是直接绘制图表，而是先将图表定义为一套完整配置，再根据配置决定：

1. 哪些内容由前端直接控制
2. 哪些内容需要向后端请求数据
3. 哪些内容需要在前端完成渲染转换

整体处理过程如下：

1. 用户选择图表模板。
2. 前端初始化图表配置。
3. 用户在设计态维护各类配置。
4. 前端将配置拆分为“前端属性配置”和“数据请求配置”。
5. 数据请求配置提交给后端，获取图表原始数据。
6. 前端根据前端属性配置和后端返回数据生成图表结果。
7. 图形类图表生成 ECharts option，表格类图表生成列定义和数据源。

## 3. 配置分类

从产品和实现角度看，本项目一共可以分为 `8 类配置`。其中一部分配置由前端直接控制，一部分配置需要参与后端请求。

### 3.1 配置分类总览

| 配置类别 | 主要内容 | 是否需要请求后端数据 |
| --- | --- | --- |
| 1. 模板配置 | 模板类型、模板能力边界、默认配置 | 否 |
| 2. 基础信息配置 | 图表标题、子标题、组件名称 | 否 |
| 3. 数据请求配置 | 数据模型、维度、指标、过滤条件、排序、条数限制 | 是 |
| 4. 指标展示配置 | 展示方式、左右轴、颜色、平滑、堆叠 | 否 |
| 5. 图层配置 | 图层名称、图层启停、图层绑定关系 | 否 |
| 6. 统计指标配置 | 统计项绑定、滚动 N 年、均值、方差区间、分位数 | 部分需要 |
| 7. 交互配置 | tooltip、legend、dataZoom、slider | 否 |
| 8. 布局配置 | 卡片位置、宽度、高度 | 否 |

### 3.2 各类配置说明

#### ① 模板配置

模板配置用于定义当前图表是什么、支持哪些能力。模板本身由前端内置管理，不依赖后端计算。

模板配置主要包括：

- 模板编码
- 模板名称
- 模板说明
- 支持的图形类型
- 允许的维度数量
- 允许的指标数量
- 是否支持图层
- 是否支持统计指标
- 默认配置

#### ② 基础信息配置

基础信息配置用于定义图表的展示名称和页面文本信息，完全由前端控制，不参与数据查询。

主要包括：

- 图表标题
- 图表副标题
- 横轴名称
- 左轴名称
- 右轴名称

#### ③ 数据请求配置

数据请求配置是所有配置中最核心的一类，它直接决定前端需要向后端请求什么数据。

主要包括：

- 数据模型
- 主维度
- 第二维度
- 系列拆分字段
- 指标列表
- 过滤条件
- 排序条件
- 条数限制

这一类配置需要由前端整理后作为请求参数传递给后端，由后端返回原始数据集。

#### ④ 指标展示配置

指标展示配置不改变后端返回什么数据，而是决定前端如何展示已经返回的数据。

主要包括：

- 指标显示名称
- 展示方式
- 左右轴归属
- 指标颜色
- 平滑开关
- 堆叠开关
- 正负值颜色规则

#### ⑤ 图层配置

图层配置用于组织图表中的逻辑展示层，由前端控制，不需要后端单独理解图层概念。

主要包括：

- 图层标识
- 图层名称
- 图层启停状态
- 维度绑定图层
- 指标绑定图层
- 统计项绑定图层

#### ⑥ 统计指标配置

统计指标配置由两部分组成：

- 一部分是前端固定属性，用于定义统计项如何展示
- 一部分需要依赖后端返回的原始指标数据，再由前端基于原始数据进行统计计算

主要包括：

- 统计项名称
- 绑定指标
- 所属图层
- 滚动窗口 N 年
- 均值线开关
- 方差区间开关
- 分位数线开关
- 样式配置

#### ⑦ 交互配置

交互配置决定图表在运行态是否显示图例、提示框和缩放能力，由前端直接控制。

#### ⑧ 布局配置

布局配置决定图表卡片在设计态和运行态中的排布方式，由前端直接控制。

## 4. 哪些能力属于前端固定属性

所谓“前端固定属性”，是指这类内容不需要后端重新计算，前端即可直接控制和生效。

### 4.1 前端固定属性总览

| 分类 | 配置项 |
| --- | --- |
| 模板属性 | 模板类型、模板说明、模板默认配置、模板能力边界 |
| 文本属性 | 标题、副标题、坐标轴名称 |
| 样式属性 | 颜色、线宽、面积透明度、点位显示、平滑 |
| 展示属性 | 折线/面积/柱状/散点/表格展示方式 |
| 坐标轴属性 | 左右轴归属、是否展示百分比轴 |
| 图层属性 | 图层名称、图层启停、图层绑定关系 |
| 交互属性 | tooltip、legend、dataZoom、slider |
| 布局属性 | 卡片位置、宽度、高度 |

### 4.2 柱状图的前端固定属性

柱状图中以下能力由前端直接控制：

| 属性 | 说明 |
| --- | --- |
| 不同指标不同颜色 | 不同指标可配置为不同柱体颜色 |
| 同一指标正负不同颜色 | 同一指标可根据正值和负值应用不同颜色 |
| 堆叠开关 | 控制柱状图是并列还是堆叠 |
| 柱体透明度 | 控制视觉效果 |

### 4.3 折线图的前端固定属性

折线图中以下能力由前端直接控制：

| 属性 | 说明 |
| --- | --- |
| 折线/面积切换 | 同一指标可切换为折线或面积形式 |
| 点位显示 | 控制是否显示节点 |
| 平滑开关 | 控制是否平滑 |
| 左右轴归属 | 控制指标落左轴还是右轴 |
| 指标颜色 | 控制不同指标颜色 |

## 5. 哪些能力需要请求后端数据

所谓“需要请求后端数据”，是指前端无法仅靠本地配置直接得出结果，必须依赖后端返回原始数据。

### 5.1 需要后端参与的内容

| 分类 | 具体内容 | 原因 |
| --- | --- | --- |
| 维度数据 | 主维度、第二维度、分类字段 | 前端本地不持有完整业务数据 |
| 指标数据 | 指标值 | 需要后端根据数据模型查询结果 |
| 过滤结果 | 过滤后的数据集 | 过滤条件应作用于后端数据源 |
| 排序结果 | 排序后的结果集 | 排序应基于后端返回结果 |
| 分页基础数据 | 表格展示数据 | 表格依赖后端返回行数据 |

### 5.2 前端向后端提交的典型请求参数

| 参数类别 | 主要参数 |
| --- | --- |
| 数据源参数 | 数据模型编码 |
| 维度参数 | 主维度、第二维度、系列拆分字段 |
| 指标参数 | 指标列表 |
| 查询条件 | 过滤条件、排序条件、条数限制 |

### 5.3 后端返回后的前端处理

后端返回数据后，前端继续完成以下工作：

1. 根据图层配置过滤当前可见维度、指标和统计项。
2. 根据指标展示配置决定当前指标显示为折线、柱状还是面积。
3. 根据颜色和样式配置生成展示属性。
4. 根据统计配置决定是否计算滚动 N 年统计结果。
5. 将结果转换为 ECharts option 或表格结构。

## 6. 统计指标如何处理

统计指标是前端方案中的重点，因为它既依赖配置，也依赖数据。

### 6.1 统计指标中的前端固定属性

以下内容由前端配置直接决定：

| 属性 | 说明 |
| --- | --- |
| 统计项名称 | 展示名称 |
| 绑定指标 | 统计项基于哪个指标计算 |
| 所属图层 | 在哪些图层显示 |
| 滚动窗口 N 年 | 统计窗口长度 |
| 是否显示均值线 | 展示开关 |
| 是否显示方差区间 | 展示开关 |
| 是否显示分位数线 | 展示开关 |
| 统计项样式 | 颜色、透明度、线型 |

### 6.2 统计指标中依赖后端数据的部分

以下内容必须依赖后端返回的原始时间序列数据：

| 数据内容 | 用途 |
| --- | --- |
| 主指标历史值 | 用于计算滚动均值 |
| 主指标历史值 | 用于计算滚动方差区间 |
| 主指标历史值 | 用于计算滚动分位数 |
| 时间维度序列 | 用于识别每个时间点的窗口边界 |

### 6.3 统计指标计算过程

统计指标的处理过程建议如下：

1. 后端返回主指标的时间序列数据。
2. 前端读取统计项配置，识别绑定指标和滚动 N 年窗口。
3. 对每一个时间点，向前截取 N 年窗口样本。
4. 如果当前时间点往前不足完整 N 年数据，则该时间点不展示统计结果。
5. 如果满足完整 N 年数据，则继续计算：
   - 滚动均值
   - 滚动方差区间
   - 滚动分位数
6. 前端将统计结果组织为额外序列。
7. 最终与主图序列一起生成 ECharts option。

## 7. 配置如何转换为 ECharts Option

图形类模板的核心过程是：配置先分类，再映射为 ECharts option。

### 7.1 转换步骤

| 步骤 | 处理内容 |
| --- | --- |
| 1 | 根据模板识别图表类型 |
| 2 | 根据图层配置过滤有效维度、指标和统计项 |
| 3 | 根据数据请求配置使用后端返回结果构造基础数据集 |
| 4 | 根据展示配置生成系列结构 |
| 5 | 根据坐标轴配置生成 xAxis、yAxis |
| 6 | 根据交互配置生成 legend、tooltip、dataZoom |
| 7 | 根据统计配置追加统计序列 |

### 7.2 配置与 Option 的映射关系

| 配置类别 | 配置项 | 对应 Option |
| --- | --- | --- |
| 数据请求配置 | 主维度 | `xAxis.data` 或时间轴配置 |
| 数据请求配置 | 指标列表 | `series` 基础数据 |
| 数据请求配置 | 系列拆分字段 | 多个 `series` 分组 |
| 指标展示配置 | 展示方式 | `series.type` |
| 指标展示配置 | 颜色 | `series.itemStyle` / `series.lineStyle` |
| 指标展示配置 | 平滑 | `series.smooth` |
| 指标展示配置 | 堆叠 | `series.stack` |
| 指标展示配置 | 左右轴归属 | `series.yAxisIndex` |
| 样式配置 | 面积透明度 | `series.areaStyle` |
| 文本配置 | 标题 | `title.text` / `title.subtext` |
| 交互配置 | 图例 | `legend.show` |
| 交互配置 | 提示框 | `tooltip.show` |
| 交互配置 | 缩放 | `dataZoom` |
| 统计指标配置 | 分位数线 | `series` + 百分比值轴 |

### 7.3 各模板对应的 Option 形式

#### 折线图

- 使用 `xAxis + yAxis + line series`
- 多个指标对应多个折线序列
- 第二维度可作为系列拆分
- 统计项追加为额外序列

#### 柱状图

- 使用 `xAxis + yAxis + bar series`
- 多个指标对应多个柱状序列
- 支持并列和堆叠
- 支持不同指标不同颜色、同一指标正负不同颜色

#### 散点图

- 使用 `xAxis(value) + yAxis(value) + scatter series`
- X 指标和 Y 指标分别映射两个数值轴
- 分类字段决定散点系列分组

#### 混合图

- 使用统一 `xAxis`
- `series` 中同时存在 `line`、`bar`、`areaStyle`
- 不同指标通过 `yAxisIndex` 分配到左轴或右轴

#### 表格

表格不生成 ECharts option，而是生成：

- 列定义
- 数据源
- 分页配置

## 8. 前端页面形式

### 8.1 设计态

设计态页面建议分为三个区域：

| 区域 | 内容 | 作用 |
| --- | --- | --- |
| 工具区 | 模板选择、组件新增 | 作为图表创建入口 |
| 画布区 | 图表卡片、布局拖拽、预览 | 作为结果预览区域 |
| 配置区 | 配置模块分区表单 | 作为图表配置入口 |

### 8.2 运行态

运行态页面重点是展示已经配置完成的图表结果。

| 能力 | 前端形式 |
| --- | --- |
| 图表展示 | 图表卡片展示 |
| 表格展示 | 表格视图或切换视图 |
| 图层切换 | 图层切换器 |
| 图表交互 | tooltip、legend、dataZoom |

## 9. 预期交付结果

| 交付项 | 交付内容 |
| --- | --- |
| 配置体系 | 明确 8 类配置及其职责边界 |
| 数据边界 | 明确哪些配置由前端控制，哪些依赖后端数据 |
| 模板体系 | 支持折线图、柱状图、散点图、表格、混合图 |
| 渲染体系 | 支持配置到 ECharts option 的统一转换 |
| 统计能力 | 支持滚动 N 年均值、方差区间、分位数，且不足 N 年不展示 |
| 页面能力 | 支持设计态配置、运行态展示和图层切换 |
